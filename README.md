# 📘 Cloud Cheatsheet

> Interactive CLI UKK By Galih Gratia Arno

## 🚀 Quick Install

```bash
curl -sL https://arno-ukk.vercel.app/cheatsheet.sh -o ~/.ukk-cheatsheet.sh
chmod +x ~/.ukk-cheatsheet.sh
bash ~/.ukk-cheatsheet.sh
```

## Hapus versi lama dan cache
```bash

```

📌 CARA PAKAI:

   1. Pilih menu (angka 1-9, A, B)
   2. Baca panduan di nano
   3. Copy command yang diperlukan
   4. Ctrl+X untuk keluar
   5. Kembali ke menu
   6. Q untuk quit

📌 KEYBOARD SHORTCUT DI NANO:

   ↑↓←→     : Navigate
   PageUp/Dn: Scroll page
   Ctrl+W   : Search keyword
   Ctrl+Home: Go to top
   Ctrl+End : Go to bottom
   Ctrl+X   : Exit to menu
   Ctrl+K   : Cut line
   Ctrl+U   : Paste
